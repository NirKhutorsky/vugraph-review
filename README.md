# VuGraph Candidate Review

Streamlit app for reviewing bridge puzzle candidates scraped from BBO VuGraph archives.

## Features

- Hand diagrams with colored suit symbols
- Quality score badges with breakdown
- Sort and filter by score, type, vulnerability, tournament
- Accept / Reject / Notes per candidate
- Export accepted candidates to JSON

## Deployment

Deployed on [Streamlit Community Cloud](https://share.streamlit.io). Password-protected.

## Data

`data/all_candidates.json` contains bundled candidates generated by the [BridgeMind vugraph scraper](https://github.com/NirKhutorsky/BridgeMind/tree/main/scripts/vugraph_scraper).
